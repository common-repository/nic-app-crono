== Description ==

Nic-app Crono is a plugin that allows you to unify different calendars (Google Calendar, Apple iCloud, Exchange, Office 365 / Outlook) into a single calendar.

Plugins developed for WordPress and WooCommerce usually work by connecting to a single calendar. What happens if you need information contained in more than one calendar?

With this plugin with the help of Cronofy you can solve this problem, keeping information from multiple sources in a single calendar.

Requirements:

php 7.1
